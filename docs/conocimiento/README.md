# Base de conocimiento de los agentes

Carpetas de material de referencia que los agentes especializados leen para cumplir su función.
Cada subcarpeta tiene un `README.md` con la lista exacta de documentos que debe contener. Coloca
aquí PDFs, plantillas en blanco y fichas de papers; no metas código (eso vive en `backend/`).

| Carpeta | La usa el agente | Contenido |
|---|---|---|
| `formatos-caces/` | `formatos-caces` | Modelo CACES 2024 + plantillas oficiales en blanco + papers |
| `indicador-3-malla/` | `indicador-3-malla` | Malla oficial de Software + ejemplos + volcados |
| `indicador-4-silabo/` | `indicador-4-silabo` | Plantilla SGC.DI.321 + sílabos ejemplo + ground truth |
| `indicador-6-guia/` | `indicador-6-guia` | Guía de laboratorio en blanco + guías ejemplo |
| `indicadores-1-2/` | `indicadores-1-2-semanticos` | Perfil de egreso + proyecto curricular + modelo educativo |
| `objetivos-tesis/` | `objetivos-tesis` | Documento de tesis + anteproyecto + rúbrica + cronograma |
| `evaluacion/` | `evaluacion-pruebas` | Ground truth, resultados de evaluación y notas de método |

> Nota de privacidad: parte de este material (rúbricas, actas, documentos institucionales) puede
> ser sensible. Está bajo `docs/` del repo: revisa el `.gitignore` antes de commitear si no debe
> publicarse.

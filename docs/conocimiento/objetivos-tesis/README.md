# Conocimiento — objetivos-tesis

Material para alinear el trabajo con los objetivos y preparar la defensa. Coloca aquí:

- `documento_tesis.pdf|docx` — el documento de tesis vigente (el capítulo de objetivos manda
  sobre `docs/CONTEXTO_TESIS.md`; si difieren, avisa).
- `anteproyecto.pdf` — anteproyecto aprobado (objetivos General/OE1/OE2/OE3 originales).
- `rubrica_evaluacion_tesis.pdf` — criterios con los que se calificará la tesis.
- `cronograma.pdf|xlsx` — fechas y entregables (para convertir fechas relativas en absolutas).
- `actas_tutor/` — actas/observaciones del tutor (decisiones acordadas, cambios de alcance).
- `resultados_evaluacion/` — copia de las salidas del OE3 cuando existan (matrices de confusión,
  kappas, Jaccard, tablas de recursos), que el sistema genera en `backend/data/resultados_evaluacion/`.

> Antes de dar un % de cumplimiento del OE3, mide lo que ya existe: `scripts/banco_pruebas.py`,
> `evaluar_campos.py`, `evaluar_jaccard.py`, `completitud.py`. NO repitas el "~0%" viejo sin medir.

# Conocimiento — formatos-caces

Material que sustenta las decisiones de formato, por sus dos fuentes: el **modelo CACES 2024** y
la **literatura académica**. Coloca aquí:

## 1. Modelo de evaluación CACES 2024 (obligatorio)
- `modelo_caces_2024.pdf` — documento oficial del modelo de evaluación (Ecuador), o al menos el
  capítulo de la **dimensión Currículo** con sus indicadores, estándares y elementos fundamentales.
- `matriz_indicadores.pdf|xlsx` — la matriz/rúbrica de indicadores si existe por separado.
> Es la fuente contra la que se valida `backend/data/caces_2024_oficial.txt`.

## 2. Plantillas oficiales EN BLANCO (obligatorio para los instrumentados)
- `plantilla_silabo_SGC.DI.321.pdf` — formulario de sílabo vacío (indicador 4).
- `plantilla_malla.pdf` — formato oficial de malla curricular (indicador 3).
- `plantilla_guia_laboratorio.pdf` — "GUIA DE USO DE LABORATORIO" vacía (indicador 6).
- (opcional) `plantilla_perfil_egreso.*`, `plantilla_proyecto_curricular.*` — si se instrumentan 1/2.

## 3. Papers académicos (sustento del OE1)
Organiza por tema; cada uno como PDF o ficha `.md` con **DOI/URL, autor y año**:
- `papers/acreditacion_universitaria/`
- `papers/gestion_evidencias_documental/`
- `papers/llm_rag_clasificacion_documentos/`
- `papers/rpa_ipa_automatizacion/`
- `papers/BIBLIOGRAFIA.md` — índice con la cita completa de cada uno (para no inventar referencias).

## Qué NO va aquí
Documentos llenos de prueba (esos van a las carpetas por indicador) ni código.
